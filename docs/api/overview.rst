API reference
=============

There are two interfaces for Psifr. The high-level :py:mod:`~psifr.fr` interface is designed to work with whole datasets in table format. The lower-level :py:mod:`~psifr.stats` interface operates on study and recall list information for individual participants.

The :py:mod:`~psifr.measures` and :py:mod:`~psifr.maskers` modules provide tools to make it easier to implement individual statistics and analyses.
